## chess-isofit
This repository contains the code used to produce custom surface reflectance data products for the Colorado Headwaters Ecological Spectroscopy Study (CHESS). We,

1. derive a set of vicarious radiometric calibration coefficients (rccs),
2. perform atmospheric correction for each flightline using isofit,
3. calculate ewt from reflectance,
4. calculate shade from a digital surface model using a ray-tracing approach,
5. mosaic the flightlines for each domain and year,
6. validate the reflectance retrievals against in situ reflectance data, and
7. extract reflectance, uncertainty, and shade for a set of polygons.

The resulting datasets are published at:

Carroll E ; Goulden T ; Chadwick K D ; Ade C ; Eckert R ; Brodrick P ; Breckheimer I (2026): Custom surface reflectance, shade mask, and equivalent water thickness mosaics for the Colorado Headwaters Ecological Spectroscopy Study (2018). Watershed Function SFA, ESS-DIVE repository. Dataset. doi:10.15485/3013527 accessed via https://data.ess-dive.lbl.gov/datasets/doi:10.15485/3013527
